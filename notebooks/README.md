# Notebooks

Download the dataset from [Google Drive](../README.md#dataset) into `data/raw/` and `data/processed/` before running notebooks that load CSV files.

| # | Notebook | Description |
|---|----------|-------------|
| 01 | `01_data_preprocessing.ipynb` | Raw CSV → cleaned dataset |
| 02 | `02_vehicle_health_eda.ipynb` | Exploratory data analysis |
| 03 | `03_isolation_forest.ipynb` | Isolation Forest training |
| 04 | `04_lstm_autoencoder.ipynb` | LSTM AutoEncoder training |
| 05 | `05_fuzzy_logic_fusion.ipynb` | Fuzzy fusion & evaluation |
